# 前端

## 技术框架

* vite+vue3 搭建项目
* docker 打包项目

### 本地搭建过程

1. 先去官网下载对应系统的node.js  

2. 从git仓库拉取``` git clone xxx ```

3. 进入bigc-answer-front文件夹terminal输入  ``` npm install ```*可以更换成阿里源*

4. 启动项目 ```npm run dev```
